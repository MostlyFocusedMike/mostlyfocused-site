What is this site?

I've been writing technical articles for over 6 years now, and I think it's time I started getting serious. Here is where I will publish all future articles as well as republishing all of the old ones from medium. I hope to have a lot more creature comforts here like:

  • Syntax highlighting
  • Code copy buttons
  • Section and code jumping
  • Find articles by topics
  • Faster load times

And more! Probably! Being 100% controlled by me means that this site can do darn near anything I want it to. Eventually I want to be fully PWA compliant as well.

But for more info about the technical side, check out my article where I talk about this site.

I'm making a newsletter!

The first Tuesday of the month, I'll send out an email to all my subscribers. It will feature my newest articles, but more importantly I'm going to include a lot of other resources that I've found as well. It's a great way to support this site and it's only once a month. Why not give it a shot?
